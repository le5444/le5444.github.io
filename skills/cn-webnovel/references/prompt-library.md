## 一、开局提示词

### 1.1 通用黄金三章生成
```
请按照以下模板为我写一本[类型]小说的黄金三章（共约6000-9000字）：

【主角设定】
姓名：[主角名]
年龄：[年龄]
初始身份：[身份]
隐藏身份/潜力：[隐藏设定]
金手指：[金手指描述]
性格关键词：[3-5个关键词]

【第一章要求】
- 200字内进入强冲突（被退婚/被追杀/被开除/被背叛/濒死...）
- 通过冲突自然带出主角背景
- 1000字内获得金手指
- 展示金手指的第一次使用效果
- 章末留下强烈悬念
- 每段不超过150字，大量短段落

【第二章要求】
- 金手指价值确认（第一次实质收益）
- 引入第一个短期目标（3-5章内可达成的目标）
- 设置第一个小反派（铺垫后续打脸）

【第三章要求】
- 第一次打脸/震惊场景
- 第一次实质性收获（资源/人脉/地位提升）
- 章末埋更大的伏笔

请开始创作。
```

### 1.2 废柴逆袭流开局
```
请以"废柴逆袭"流为我写小说开篇第一章（3000字）：

【设定】
主角：[姓名]，[年龄]，[宗门/家族]公认的废物，无法修炼的废脉
金手指：在[绝境]中意外获得[神秘传承/太古至宝/系统激活]
对手：曾经的手下败将反过来欺辱主角
期待点：让读者急不可耐想看主角用金手指翻身打脸

【必须包含的元素】
- 开篇被嘲讽/羞辱的具体场景
- 主角内心的不甘和屈辱
- 绝境中金手指降临的震撼瞬间
- 金手指第一次使用的爽感
- 对即将开始的打脸之路的强烈暗示
```

### 1.3 重生逆袭流开局
```
请为我写重生小说的开篇：

【设定】
主角前世：[前世最高成就] 却 [前世最惨结局——被谁背叛/害死]
重生节点：回到 [具体时间点——如高考前三天/末世降临前一周/被退婚当天]
前世记忆觉醒的触发：[触发方式]

【必须包含】
- 前世惨死的画面（简短但冲击力强）
- 重生后的震撼和不敢置信（适度，不要过度感叹）
- 确认重生后的第一个行动（利用先知先觉做一件事，制造第一个期待点）
- 对前世的仇人的描写（为复仇线铺垫）
- 主角的心态转变（这次，我要改写一切）
```

### 1.4 开局引子——第一章前200字模板

**模板A：绝境开场**
```
刀锋贴在脖子上，冰凉的触感让[主角名]浑身僵住。
"废物就是废物。"[反派名]的声音带着毫不掩饰的轻蔑，"三年了，连炼气一层都没突破，你有什么资格留在这？"
周围的师兄弟们发出一阵哄笑。
[主角名]攥紧了拳头。指甲深深地嵌进掌心。
但他没有再争辩。
因为他们说的是对的。
他的确是个废物。
——只是所有人都不知道，他胸口的那块玉佩，正在微微发烫。
```

**模板B：退婚开场**
```
"这是退婚书，签了吧。"
一纸信笺被随手扔在[主角名]面前。
他抬起头，对上[女主名]冷若冰霜的脸。三年前，他经脉尽断、修为全废的那天，她就再也没对他笑过。
"我..."
"不要叫我。"她打断他，"前世的婚约已不作数。从今往后，你我各不相欠。"
满堂的人都看着他。
看一个废物如何被未婚妻休掉。
——没有人注意到，他丹田深处，那沉睡了千年的太古血脉正在苏醒。
```

**模板C：系统降临开场**
```
[主角名]觉得自己的运气从来没有这么差过。
被公司开除、女朋友跟人跑了、连房租都付不起——然后从天而降的一个花盆砸在他头上。
视线模糊的最后一秒，他听到一个机械的声音在脑海中响起：
【宿主生命体征下降...检测中...】
【滴——万界签到系统激活。】
【新手签到地点：和平小区垃圾站。】
【签到奖励：天级功法《混元诀》。】
```

## 二、打脸场景提示词

### 2.1 身份揭露型打脸
```
请为我写一个"隐藏大佬身份揭露"的打脸场景：

【场景设定】
地点：[如：高档酒店/拍卖会场/家族宴会]
在场人物：
- 主角：[姓名]，表面身份是[卑微身份]，真实身份是[惊人身份]
- 反派：[姓名]，[身份]，正在当众羞辱/轻蔑主角
- 关键人物：[姓名]，[身份]，即将揭示主角真实身份的人
- 围观群众：[描述]

【氛围要求】
- 反转前：反派的嚣张、围观者的附和、主角的淡然（不是怂，是不屑）
- 反转时：关键人物的出场要震撼（大门被推开/车队抵达/全体起立）
- 反转后：多层次反应——反派的恐惧、围观者的震惊、主角的从容

【台词要求】
- 反派嘲讽的话要足够嚣张（铺垫越足，打脸越爽）
- 关键人物揭示身份的话要有重量感
- 主角最后的话要轻描淡写但杀人诛心
```

### 2.2 实力碾压型打脸
```
场景：主角被挑衅后展露实力碾压所有人
要求：
1. 挑战者要先展示"最强一击"（让读者觉得确实很强）
2. 主角轻描淡写地挡住/破解
3. 多角度描写围观者的震惊反应
4. 主角打完后的态度：不值一提/还要继续吗/我来不是跟你们玩的
5. 这场战斗结束后要改变主角在群体中的地位
```

### 2.3 知识/专业碾压型打脸
```
场景：主角在专业场合被质疑，用远超所有人的专业知识碾压
适用类型：医生文/鉴宝文/学术流/商战文

要求：
1. 设置一个"权威"角色先给出错误判断
2. 主角指出问题，被众人嘲笑
3. 主角用无可辩驳的证据/操作证明自己是对的
4. "权威"面子扫地，主角被刮目相看
5. 不止是知识碾压——这场胜利要带来资源/人脉/名声的实际收益
```

## 三、战斗场景提示词

### 3.1 一对多碾压战斗
```
请写一场主角一人碾压多人的战斗场景：

【战斗背景】
对手：[人数]人，[实力等级]，[特殊能力]
主角：[当前实力]，[核心战斗方式]
战斗目的：[保护某人/立威/争夺资源/复仇]

【写作要求】
- 用短句和动词制造节奏感
- 每个对手的倒下方式要有变化（不能千篇一律）
- 穿插围观者的反应
- 主角可以受伤但要帅气
- 战斗结束后让围观者重新评估主角的地位
```

### 3.2 势均力敌的激战
```
请写一场势均力敌的战斗：

要求：
1. 双方都要展示实力（让对方成为好的对手）
2. 三回合交锋：试探→全力交锋→绝招对决
3. 主角在战斗中突破/领悟新能力（战斗中成长）
4. 胜利要有代价（受伤/消耗/牺牲）
5. 战斗结束后要有战略层面的影响（局势变化）
```

### 3.3 越级战斗
```
请写一场主角越级斩杀强敌的战斗：

核心要求：
1. 明确双方的等级差距（数字化的实力对比）
2. 主角不能纯靠蛮力越级——要用智慧+环境+对方弱点+金手指
3. 要有命悬一线的紧张感
4. 胜利的过程要让读者觉得"合理"——赢得惊险但服气
5. 越级战斗的代价（消耗寿元/重伤/金手指休眠一段时间）
6. 战后各方反应——越级斩杀的战略意义
```

## 四、金手指设计提示词

### 4.1 系统流金手指完整设计
```
请为我设计一个完整的系统流金手指：

【基本属性】
系统名称：[名字]
绑定方式：[触发条件]
核心功能：[功能1]、[功能2]、[功能3]
能量/货币：[积分/点数/能量]的获取方式

【成长体系】
第一阶段（前100章）：基础功能 [功能列表]
第二阶段（100-300章）：解锁 [进阶功能]
第三阶段（300-500章）：解锁 [高级功能]
隐藏功能：[需要特定条件触发]

【限制条件】
使用代价：[代价1]、[代价2]
冷却限制：[时间/次数限制]
功能禁区：[什么情况下系统无法使用]
惩罚机制：[失败/违规的惩罚——必须具体可感知]

【主线任务】
新手任务：[任务描述] → 奖励：[奖励内容]
进阶任务（3个）：[递增难度]
终极任务：[贯穿全书的目标]

【系统来源之谜】
表面：[最初的理解]
中期线索：[发现系统并非那么简单]
真相：[系统的真正来源和目的——用作中后期大剧情]
```

### 4.2 老爷爷/随身流金手指
```
请设计一个"随身老爷爷"型金手指：

设定要求：
1. 老爷爷的身份：[前世强者/上古大能/异界灵魂/人工智能]
2. 状态：[重伤/封印/残魂/被诅咒]
3. 能力边界：[擅长什么] + [不能做什么]
4. 帮助方式：[指导修炼/传授功法/战斗时借力/信息提供]
5. 限制：[定期沉睡/消耗能量/需要主角完成任务来恢复]
6. 情感线：从导师→朋友→被保护者（角色关系随剧情演变）
7. 隐藏线：老爷爷有自己的恩怨（作为中期剧情线）
```

## 五、角色塑造提示词

### 5.1 主角完整设定
```
请为[类型]小说设计完整的主角人设：

【基础信息】
姓名：[有寓意的名字]
年龄：[出场年龄]
外貌特征：[3-5个标志性特征]
身份：[表面身份] → [真实身份] → [终极身份]

【性格体系】
核心性格（3个标签）：[标签1/标签2/标签3]
优点（3个）：[优点]
缺点/缺陷（3个）：[缺点]——包括一个致命的性格缺陷
口头禅/习惯动作：[独特性标志]
逆鳞/底线：[什么情况下主角会失控]

【成长线】
起点状态：[弱者/普通人/被废的天才]
中期成长：[能力的提升 + 心态的成熟]
终极形态：[实力的巅峰 + 理念的完成]

【情感模式】
对朋友：[态度]
对敌人：[态度]
对异性：[态度]
爱情观：[核心——如"一生一世一双人"或"我全都要"]

【深层驱动】
核心欲望（外在想要什么）：[外在目标]
核心需求（内在需要什么）：[内在需求]
最大恐惧：[什么让主角害怕]
谎言/偏执：[主角相信的关于世界的错误信念]
```

### 5.2 反派设计（三层深度）
```
请设计一个有深度不扁平的反派：

【表面层】
姓名/身份/实力：[基本信息]
对主角做的事：[具体罪行]
读者的第一印象：[为什么恨他]

【中间层】
动机：[他为什么这么做——要有合理理由]
背景故事：[是什么让他变成这样]
他与主角的镜像关系：[他和主角哪里相似，哪里相反]

【深层】
他自认为的正义：[他觉得自己的行为是合理的]
他的人性残留：[他还在乎什么——某个人/某个信念]
救赎可能性：[能否被救赎/什么情况可能改变他]

【在故事中的作用】
前期：[对主角的压迫方式]
中期：[他的势力/计划逐渐揭露]
后期：[最终对决的情感意义——不只是打架]
```

## 六、各类型专项提示词

境界2：[名称] — 特征：[...]

- 资源需求：[每境界需要什么资源/丹药]

战力划分：
同境界内：普通/精英/天才/妖孽/绝世的战力对比

修炼流派：
- [流派1]：[特点/擅长/代表人物]
- [流派2]：[特点/擅长/代表人物]

```

```

【信息收集阶段】
主角掌握的情报：[具体]

【连环计布局】

【计谋的反转点】
在执行过程中遇到的意外：[具体]
主角的应变：[展示主角的急智]

【计谋结果】
对主角的影响：[地位变化/势力变化/他人态度变化]
对后续剧情的影响：[引出更大的矛盾]
```

```

【副本基本信息】

参与人数：[人数]
通关条件：[明确规则]
失败惩罚：[具体惩罚]

## 八、情感场景提示词

### 8.1 糖分超标 — 甜宠场景
```
请写一段甜宠场景：

【人设】
男主：[霸道总裁/高冷学神]，平时对所有人[冷漠/疏离]，唯独对女主[温柔/占有欲强]
女主：[软萌/沙雕/独立]，[特征]

【场景】
[如：女主生病，男主放下重要会议飞奔回家照顾]

【要求】
- 男主的反差感要拉满（对外冷酷对内温柔）
- 细节比台词更重要（手背试体温/掖被角/默默守着）
- 不要直接说"我爱你"——用行动诠释
- 加入至少一个让读者发出尖叫的细节
```

### 8.2 虐心场景
```
请写一段虐心场景：

【要求】
- 两人明明相爱却被命运推着走向对立
- 要有"就差一点点"的感觉——差一点就能在一起
- 用细节堆积心痛感（他记得她的一切，却不能再靠近）
- 对话要有潜台词（表面说一套，心里想另一套）
- 结尾给一个读者会反复看的画面/台词
```

### 8.3 追妻火葬场
```
场景：男主之前伤害了女主，现在幡然醒悟开始疯狂追妻

要求：
1. 男主的悔恨要真实（不只是说对不起，要有行动上的巨大付出）
2. 女主不是轻易原谅——要给读者看到男主"受虐"的过程
3. 设置"比较"场景：男主看到女主身边出现了更好的人
4. 最终和解要有充分铺垫，不能突兀
5. 追妻过程中展示男主真正的改变（不只是为了追回来）
```
